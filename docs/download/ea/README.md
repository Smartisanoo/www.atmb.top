# 2020.8版本进入早期测试阶段
在将来的至少一年内,2020.8将作为主要支持版本存在。

## 主要变化
* 使用HandyControl重做界面
* 大量底层逻辑修改
* 使用AppCenter进行跟踪
* 使用.Net Core 3.1作为框架,并且自带完整运行时，无需单独下载。
* 基础的SDK 11.7实现
