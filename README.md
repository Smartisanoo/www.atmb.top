# 秋之盒官网开源
## AutumnBox Official Website Open Source
[https://www.atmb.top](https://www.atmb.top)
## 运行环境
* node.js >= 12
## 下载
```sh
#下载代码仓库到本地
git clone git@github.com:zsh2401/www.atmb.top.git atmb-top-website

```
### 安装依赖
```
#进入目录
cd atmb-top-website && yarn
```
### 运行调试
```
yarn dev
```
### 构建静态文件
```
yarn build
```
