## 运行环境
node.js >= 12
## 下载
```sh
git clone git@github.com:zsh2401/www.atmb.top.git && cd www.atmb.top
```
### 安装依赖
```sh
yarn #or npm install
```
### 运行调试
```sh
yarn dev #or npm run dev
```
### 构建静态文件
```sh
yarn build #or npm run build
```
