激活脚本优化软件

本页面作者: zsh2401

什么是脚本优化软件?有哪些?干什么用?

啊,其实这个说法也很笼统,"脚本优化软件"是我自创词汇.

通过ADB在手机上运行脚本,可以获得介于root和普通用户之间的权限,这个权限不多不少,正好可以用来休眠软件,而黑域凭借这样的方式,在2016年火爆酷安

这种激活方式相比DPM简单许多,几乎没有限制,但没有什么东西是完美的,通过脚本的方式激活软件DPM模式是具有时效性的,每当你重启手机后,就需要重新激活一遍,如果你经常可以用到电脑或很少关机,那么这个方式无疑是最棒的.

除黑域外,还有许多通过该方式激活的软件,他们不一定都是休眠工具

  • 小黑屋(麦克斯韦妖模式)
  • ShizukuManager
  • AppOps
  • ...

这类软件使用秋之盒激活是真正的无脑操作

激活黑域

  1. 下载并安装黑域
  2. 在手机上打开一次黑域主界面
  3. 在秋之盒中激活黑域
  4. OK!

激活小黑屋麦克斯韦妖模式

特别推荐!

  1. 下载并安装小黑屋
  2. 在手机上启动小黑屋,选择麦克斯韦妖模式
  3. 在秋之盒中激活小黑屋麦克斯韦妖模式
  4. 完美!

激活ShizukuManager

  1. 下载并安装ShizukuManager
  2. 在手机上启动ShizukuManager
  3. 在秋之盒中激活ShizukuManager
  4. Perfect!

激活AppOps

  1. 下载并安装AppOps
  2. 下载并安装ShizukuManager
  3. 激活ShizukuManager(AppOps特权模式基于ShizukuManager)
